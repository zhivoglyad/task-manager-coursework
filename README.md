# Task Manager

Итоговая курсовая работа — веб-приложение для управления задачами с поддержкой drag-and-drop, статистикой и тёмной темой.

## Стек технологий

- **Vite** + **React 18** + **TypeScript**
- **Tailwind CSS v3** — стилизация
- **Zustand** — управление состоянием
- **@dnd-kit** — drag-and-drop
- **Recharts** — графики и статистика
- **Framer Motion** — анимации
- **date-fns** — работа с датами
- **lucide-react** — иконки
- **Vitest** + **Testing Library** — тестирование

## Запуск проекта

```bash
pnpm install
pnpm dev
```

## Тесты

```bash
pnpm test
```

## Сборка

```bash
pnpm build
```
